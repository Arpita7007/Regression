# House Price Prediction using Regression

## 📌 Project Overview
This project predicts house prices using machine learning regression techniques.

## 📊 Dataset
- housing.csv
- Features include: location, income, bedrooms, etc.

## ⚙️ Steps Performed
- Data cleaning (handled missing values)
- Feature scaling (StandardScaler)
- Model training (Linear Regression)
- Model evaluation

## 📈 Results
- Model trained successfully
- Predictions vs Actual plotted

## 🛠️ Tech Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
